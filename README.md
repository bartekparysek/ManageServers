# ManageServers ⚙️

A administration panel to control status of multiple servers at the same time. It gives abbility to turn on/off and reboot servers based on API calls.

## Technologies Used

-  React - version 16.9.0
-  Styled Components - version 5.3.0
-  Testing Library - version 7.31.2
-  Axios - version 0.21.1
-  MSW - verision 0.29.0

## Features

-  List all the servers
-  Turn on server
-  Turn off server
-  Reboot server
-  Search for servers

## Setup

### UI

In `./ui` run:

```shell
npm install
```

In `./ui` run:

```shell
npm start
```

### API

In `./api` run:

```shell
npm install
```

In `./api` run:

```shell
npm start
```
